사건의 발단

AWS 로 업로드 할 때 permission denied 로 업로드 불가 😱

권한 부여하면 만고 땡임

해결방법 cmd

CentOS:

Ubuntu:

$ sudo chown -R ubuntu:ubuntu /var/www/html
$ sudo chmod -R 755 /var/www/html

Amazon Linux:

$ sudo chown -R ec2-user:ec2-user /var/www/html
$ sudo chmod -R 755 /var/www/html
