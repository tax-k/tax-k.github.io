React.js

왜 열광 하는가?

  1. Javascript 베이스에요.

    딴거 배울거 없어요. Javascript 만 부수면 됩니다.

  2. Composition

    Component 단위로 부수면 됩니다. 재사용성 겁나 좋아져요.

    JSX (리액트로 html 쓰는 법)를 이용해요.

    요렇게 쓰는 거에요.

  3. Uni-Directional Data Flow

    데이터는 항상 한 곳에 있고, 거기서만 변경 가능해요.

    데이터가 UI를 변경시키는 것. UI에서 데이터 변경 ❌
    Model -> View

    반면에 Angular 는 Bi-Directional(View <-> Model)이에요.

  4. 생태계

    글로발 남바완 커뮤니티

Webpack?

Trnasfiler

React 코드를 브라우저가 이해할 수 있는 코드로 변경 해줘요.

파일을 바꾸거나 (추가/삭제 등등) 변경이 일어날 때 자동으로 변경 해줍니다.

즉 최신 자바스크립트 버전인 ES6 를 자동으로 브라우저가 이해하게 슥 도와줘요.
